# 中国气象局国家气象中心 (NMC) 预警 API

## API 概述

**数据源**: 中国气象局国家气象中心 (National Meteorological Center)  
**官方网址**: http://www.nmc.cn  
**API 类型**: REST API (JSON)  
**认证方式**: 无需认证，完全免费公开  
**数据权威性**: 国家级气象灾害预警，由专业气象专家研判发布

## 核心接口

### 1. 查询预警列表

```bash
curl -s "http://www.nmc.cn/rest/findAlarm?pageNo=1&pageSize=50&signaltype=&signallevel=&province="
```

**参数**:
- `pageNo`: 页码（从1开始）
- `pageSize`: 每页数量（建议50-100）
- `signaltype`: 预警类型过滤（空=全部）
- `signallevel`: 预警等级过滤（空=全部）
- `province`: 省份代码（空=全国，"65"=新疆，"11"=北京）

**返回数据结构**:

### 2. 按省份过滤

## 行政区划代码结构

**alertid 前缀规则**:
- 省级预警: 6位（如 `650000` = 新疆）
- 地市级: 6位（如 `652800` = 巴音郭楞蒙古自治州）
- 县级: 6位（如 `652828` = 尉犁县）

**精准过滤逻辑**:
```python
# 匹配巴州（6528）及尉犁县（652828）的预警
target_prefixes = ["652828", "6528"]
for a in all_items:
    code = a["alertid"].split("_")[0]
    is_match = any(code.startswith(p) for p in target_prefixes)
    is_province = code.startswith("65") and len(code) <= 6
    if is_match or is_province:
        alerts.append(a)
```

## 预警类型完整列表

暴雨、大风、高温、寒潮、雷电、冰雹、沙尘暴、大雾、霜冻、干旱、地质灾害、雷暴大风、山洪灾害等。每种有蓝色/黄色/橙色/红色四级。

## 预警等级说明

| 等级 | 颜色 | 含义 | 管理建议 |
|------|------|------|---------|
| 蓝色 | 🔵 | 一般气象灾害 | 注意天气变化，调整作业计划 |
| 黄色 | 🟡 | 较重气象灾害 | 关注预警更新，做好防范准备 |
| 橙色 | 🟠 | 严重气象灾害 | 停止喷药、飞防等户外作业 |
| 红色 | 🔴 | 特别严重气象灾害 | 立即启动应急预案，停止所有户外作业 |

## 预警类型解析代码

```python
import re

title = "新疆巴音郭楞蒙古自治州气象台发布大风黄色预警信号"
m = re.search(r'发布(.+?)预警', title)
warn_type = m.group(1) if m else "未知"  # "大风黄色"

level_map = {"蓝色": "blue", "黄色": "yellow", "橙色": "orange", "红色": "red"}
level = "unknown"
for cn, en in level_map.items():
    if cn in warn_type:
        level = en
        break
```

## 数据更新频率

- **更新周期**: 实时发布，通常每小时更新
- **数据延迟**: < 5分钟
- **有效期**: 预警发布后通常有效24-48小时

## 优势与局限

**优势**:
- ✅ 官方权威，具有法律效力
- ✅ 完全免费，无调用限制
- ✅ 全国覆盖，无遗漏
- ✅ 实时更新，< 5分钟延迟
- ✅ 预警类型完整，四级等级明确

**局限**:
- ❌ 无历史查询（只能查当前有效预警）
- ❌ 无预报功能（只提供已发布的预警）
- ❌ 行政区划依赖（需知道目标区域代码）

## 与其他数据源的关系

**NMC 预警 vs 商业API预警**:
- NMC 是官方源头，具有法律效力
- 和风天气/彩云天气的预警数据来源于中央气象台，但基础套餐常返回 403

**NMC 预警 vs 数值预报**:
- NMC 预警由专业气象专家研判，综合考虑多种因素
- 数值预报只提供原始数据，**不能用于自行判断灾害**
- 自行根据阈值判断（如"风速>40km/h就算大风"）不专业，可能误报或漏报

## 最佳实践

1. **优先使用 NMC 官方预警**: 灾害预警必须来自官方权威渠道
2. **不要自行判断灾害**: 数值预报只用于参考，不能替代专业预警
3. **按行政区划过滤**: 使用正确的行政区划代码精准查询
4. **展示完整信息**: 预警类型+等级+发布时间+管理建议
5. **无预警时明确说明**: "当前无气象灾害预警"比沉默更好
6. **定期更新**: 每小时查询一次，确保预警信息及时
